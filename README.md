# Kinky Wochenplan

Eine einfache Browser-Version für einen geplanten Wochenüberblick im dunklen, pinken Look. Gedacht als Mischung aus Stundenplan und Wochen-To-do-Liste.

## Funktionen

- Aufgabenliste mit eigenen Einträgen
- Kategorien: Wunsch, Aufgabe, Disziplin
- Wochenplan von Montag bis Sonntag
- Drei Slots pro Tag
- Ausgabe als kopierbarer Text
- Lokale Speicherung im Browser per `localStorage`
- GitHub Pages tauglich, da nur eine einzelne HTML-Datei

## Nutzung lokal

Einfach `index.html` im Browser öffnen.

## Nutzung über GitHub Pages

1. Neues Repository anlegen
2. `index.html` hochladen
3. Optional `README.md` ergänzen
4. In den Repository-Einstellungen GitHub Pages aktivieren
5. Die Seite wird direkt als statische Website ausgeliefert
